# 🎨 백프로: 종합 위험 점수 대시보드 디자인 브리프 (v1.0)

## 🎯 1. 프로젝트 개요
*   **목표:** 축산인 사용자에게 '데이터 나열'이 아닌, 즉각적인 '행동 가능한 인사이트'를 제공하여 의사결정을 지원하는 대시보드 구축.
*   **최종 산출물 포맷:** 고화질 Mockup (JPEG/PNG) 및 Figma 원본 파일.
*   **타겟 사용자:** 축산 농가 관리자 (B2B, 전문 지식 필요).

## 🎨 2. 비주얼 시스템 (Visual System)
### 2.1. 컬러 팔레트 (Color Palette)
| 이름 | 코드 (HEX) | 용도 | 설명 |
| :--- | :--- | :--- | :--- |
| Primary (메인) | `#0056B3` | CTA, 헤더, 강조 요소 | 신뢰감, 전문성 (짙은 블루 계열) |
| Secondary (보조) | `#6c757d` | 배경 요소, 구분선 | 중립적, 정보 구분 |
| Success (저위험) | `#28a745` | 점수/요소: 낮음 (Low Risk) | 안전, 안정적 (녹색 계열) |
| Warning (경고) | `#ffc107` | 점수/요소: 보통 (Medium Risk) | 주의, 모니터링 필요 (노란색 계열) |
| Danger (고위험) | `#dc3545` | 점수/요소: 높음 (High Risk) | 위험, 즉각적 행동 필요 (빨간색 계열) |
| Background | `#f8f9fa` | 전체 배경 | 깨끗하고 가독성 높은 배경색 |

### 2.2. 타이포그래피 (Typography)
*   **폰트 패밀리:** Noto Sans KR 또는 Pretendard (가독성 최우선)
*   **제목 (H1/H2):** 24px Bold
*   **본문/데이터:** 14px Regular (최소 16px 이상 권장)
*   **데이터 강조:** 18px SemiBold

## 🖼️ 3. 레이아웃 및 컴포넌트 정의 (Layout & Component)
### 3.1. 전체 구조 (Layout)
*   **구조:** 3단 그리드 시스템 기반의 대시보드 레이아웃.
*   **핵심 섹션:**
    1.  **상단 요약 (Hero/Summary):** 종합 위험 점수(가장 크게), 전체 요약 지표 3개 배치.
    2.  **위험 요소 시각화 (Core Visualization):** 레이더 차트 또는 게이지 차트 (3요소 비교).
    3.  **행동 매뉴얼 (Actionable Insight):** 점수별/요소별 구체적인 행동 지침 텍스트 박스.
    4.  **세부 트렌드 (Detailed Trend):** 시간 경과에 따른 주요 위험 요소 변화 그래프 (Line Chart).

### 3.2. 핵심 컴포넌트 사양
*   **[점수 시각화]**: 레이더 차트 사용. 3가지 요소가 모두 낮은 점수(Success) 영역에 위치할 때의 '이상적 상태'를 시각적으로 보여주고, 현재 점수와의 격차를 강조.
*   **[데이터 카드]**: 그림자(Soft Shadow)를 적용하여 정보의 분리도를 높이고, 각 카드 상단에 **위험 레벨(Danger/Warning/Success)** 뱃지를 명확하게 배치.
*   **[CTA (Call To Action)]**: 가장 중요한 행동 지침 버튼은 Primary Color를 사용하고, 명확한 문구(예: '사료 변경 검토', '수의사 상담 요청')와 함께 배치.
