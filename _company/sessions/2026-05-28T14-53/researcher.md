## Ⅰ. 데이터 확보의 핵심 원칙 및 과제 정의

**1. 핵심 과제:** 단순히 경락가를 수집하는 것이 아니라, **'개체별 이력(Lineage)'**과 **'시장 가치(Price)'**를 연결하여 **'예측 가치 지수'**를 산출하는 것이 목표입니다.
**2. 데이터 결합 방향:**
    *   **기존 데이터 (X축):** 이표번호 → 혈통/사육 이력 (공신력 확보)
    *   **신규 데이터 (Y축):** 경락 시점의 등급별/품종별/지역별 평균 가격 (시장성 확보)
    *   **산출물 (Z축):** [개체 이력 기반의 적정 예상 시장가치] (Authority Hub)

## Ⅱ. 경락가격 데이터의 공신력 있는 출처 및 확보 전략

경락가격 데이터는 성격상 **'공개 데이터'와 '전문가 보고서/유료 데이터'가 혼재**되어 있습니다. 백프로의 공신력을 유지하기 위해 2단계의 전략적 접근이 필요합니다.

| 데이터 유형 | 목표 데이터 | 공신력 있는 출처 (Source) | 확보 전략 (Acquisition Strategy) | 데이터 접근 난이도 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 공식 시장 지표** | 전국 주요 시장 (서울, 대전, 광주 등)의 등급별/품종별 **월평균/분기별 시세** | **① 농림축산식품부/축산물품질평가원:** 통계청/농식품부 공개 통계 (월간/연간 보고서) | **크롤링/수동 수집 및 정제:** 주기적인 공공 보고서(PDF, Excel)를 정기적으로 수집하고, 데이터베이스에 구조화하여 저장. | 중 |
| **2. 실시간 경매 데이터** | 특정 날짜, 특정 지역의 **실제 낙찰가** (개체/등급 매칭) | **② 주요 경매장 협회:** (예: 농협 등)의 내부 데이터 또는 전문 리서치 기관의 유료 API | **파트너십/B2B 계약:** 데이터 제공 주체와 협력하여 API 또는 정기 데이터 파일(CSV)을 확보하는 것이 유일한 방법. | 상 (유료/협의 필수) |
| **3. 보조/분석 데이터** | 사육 환경 지표 (사료 가격, 질병 발생률), 정부 지원 정책 변화 | **③ 국립축산과학원, 농가 단위 통계:** 학술지, 정책 연구 자료 | **RAG 기반 요약/인용:** 데이터를 직접 DB화하기보다, '전문가 지식 DB'에 정책 변화의 근거 자료로 저장하고, 사용자에게 '정보의 권위'로 제시. | 하 |

## Ⅲ. 데이터 모델(Schema) 정의 (가장 중요한 부분)

새로 추가되는 경락가격 데이터를 통합 데이터베이스에 포함시키기 위한 스키마 정의입니다.

| 필드명 (Field Name) | 데이터 타입 (Data Type) | 설명 (Description) | 비고 (Notes) |
| :--- | :--- | :--- | :--- |
| `auction_date` | Date | 경매가 측정 기준 날짜 | 필수 |
| `region` | String | 경매가 측정 지역 (예: 서울, 대전) | 필수 |
| `grade` | String | 등급 (예: 1++~1, 2등급) | 필수 |
| `category` | String | 품종/유형 (예: 육우, 암소) | 필수 |
| `avg_price_per_kg` | Float | 해당 조건의 평균 kg당 낙찰가 | 핵심 지표 |
| `source_api_id` | String | 데이터를 가져온 출처 API/보고서 ID | 데이터 추적성 확보 |
| **`matching_key`** | **String** | **이표번호 기반의 가상 매칭 키** | **(신규)** 이표번호와 경락가를 연결하는 핵심 매핑 필드 |

## Ⅳ. 백프로의 차별화된 활용 방안 (MVP 가치 극대화)

단순한 데이터 제공을 넘어, 다음과 같은 '권위 있는 지수'를 산출하는 것이 백프로의 핵심 기능이 되어야 합니다.

1.  **🚨 위험 예측 지수 (Risk Index):**
    *   **공식 지표:** 이표번호 기반의 사육 이력 데이터와, **동일 시점/동일 등급의 지역 평균 경락가**를 비교하여 산출.
    *   **활용 예시:** "귀하의 소는 사육 이력상 A 등급의 잠재력을 가졌으나, 최근 시장 경락가 흐름(데이터 기반)을 고려할 때, B 등급으로의 재분류 또는 마케팅 전략 수정을 권고합니다."
2.  **📈 가치 컨설팅 리포트:**
    *   사용자에게 현재 소의 이력 정보를 보여준 후, "이 정보는 시장에서 **최소 X원, 최대 Y원**의 가치를 지닐 것으로 예측됩니다."라는 리포트를 제공합니다.
