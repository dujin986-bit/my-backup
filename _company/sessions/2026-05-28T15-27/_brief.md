# 📋 작업 브리프

**원 명령:** 그럼 다음단계로 데이터 통합 아키텍처 설계와, 실제 프로토타입 화면 설계부터 진행해보자. 모두 움직이자 제군들

## 요약
데이터 통합 아키텍처 설계와 실제 프로토타입 화면 설계를 다음 단계의 핵심 목표로 설정함. 이 목표 달성을 위해 기술 구조 설계와 사용자 경험(UX) 시각화 작업을 동시에 진행해야 함.

## 분배
- **💻 코다리**: 현재까지 정의된 데이터 통합 모델(혈통, 사육, 도축, 기후/경제/질병 위험 지수 등)을 기반으로, '종합 위험 점수'를 산출하는 백엔드 아키텍처 설계(API Endpoint 정의, 데이터 흐름도, 필수 로직 스펙)를 완성해 주세요. 특히, 여러 이질적인 데이터를 하나의 점수로 통합하는 로직의 구체적인 수학적/기술적 구조를 제시해야 합니다.
- **🎨 Designer**: 개발된 '종합 위험 점수'와 3가지 위험 요소(기후, 경제, 질병)를 농가 단위 사용자에게 직관적으로 보여줄 수 있는 대시보드 프로토타입 화면(Wireframe)을 설계해 주세요. 점수와 각 위험 요소를 시각화할 그래프 유형(예: 레이더 차트, 게이지), 컬러 코딩(Color Coding), 그리고 사용자 행동 유도(CTA) 요소를 포함해야 합니다.
