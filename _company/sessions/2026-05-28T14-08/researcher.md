# 📈 경쟁사 벤치마킹 보고서: 키우소 어플 분석 (가안)

**타겟 청중:** 축산인
**분석 기준:** 시장 트렌드, 기능적 강점(Strength), 약점(Weakness), 기회(Opportunity)

| 트렌드 키워드 | 내용 요약 (키우소 어플 분석 결과 대기) | 백프로 적용 기회 (Opportunity) |
| :--- | :--- | :--- |
| **디지털화 및 데이터 연동** | (예: 사육 데이터 기록, 사료 원가 자동 계산 등) | 단순 정보 제공을 넘어, **개인화된 데이터 기록 및 분석 기능**을 핵심으로 설계. |
| **커뮤니티 기반 지식 공유** | (예: 특정 질병 문의, 사료 추천 등 Q&A 기능) | **익명성 기반의 전문 Q&A 포럼**을 구축하고, 질문에 대한 '검증된 답변' 시스템 도입. |

## 2. 기능적 강점/약점 분석 (SWOT Analysis)

| 구분 | 분석 영역 | 키우소 어플 (예상) | 백프로 MVP 적용 시 고려 사항 |
| :--- | :--- | :--- | :--- |
| **✅ 강점 (Strength)** | **정보의 깊이/전문성** | (예: 축산 관련 법규나 전문 용어 설명이 체계적임) | **'신뢰도'**를 최우선 가치로 설정. 정부 기관 자료, 학술 자료 등을 적극적으로 인용하여 정보의 권위를 확보해야 함. |
| **✅ 강점 (Strength)** | **UX/UI 및 접근성** | (예: 직관적인 대시보드 구성, 모바일 사용 편의성이 높음) | **'단일 목표 집중'** 원칙 적용. 너무 많은 기능을 담으려 하기보다, 축산인이 가장 자주 하는 핵심 행동(예: 시세 확인)에만 초점을 맞춰야 함. |
| **❌ 약점 (Weakness)** | **사용자 경험 (UX)** | (예: 기능이 너무 많아 사용자가 혼란스러움, 정보 탐색에 시간이 걸림) | **핵심 Pain Point 3가지**를 중심으로 기능 범위를 강력하게 축소하고, **'최소한의 동작으로 최대의 가치'**를 제공하는 경험을 설계해야 함. |
| **❌ 약점 (Weakness)** | **커뮤니티 활성화** | (예: 질문/답변의 질이 낮거나, 활동이 저조함) | **초기 사용자 유입을 위한 강력한 인센티브 설계** (예: 최적의 답변자에게 포인트를 제공)와, 전문가를 통한 '가이드 답변' 시스템이 필요함. |

## 3. MVP 적용을 위한 사용자 시나리오 및 핵심 기능 아이디어 (Actionable Insights)

경쟁사 분석을 통해 파악한 시장의 빈틈(Gap)과 백프로의 목표를 결합하여, 즉시 개발에 적용 가능한 3가지 사용자 시나리오와 핵심 기능을 제안합니다.

### 🎯 A. 사용자 시나리오 (User Story) 정의

| 시나리오 ID | 역할 (As a) | 목표 (I want to) | 가치 (So that) | Pain Point 해결 (Pain Point Addressed) |
| :--- | :--- | :--- | :--- | :--- |
| **US-1** | 축산농가 운영자 | 오늘 사육한 품종의 실시간 시장 시세를 여러 경로에서 비교하고 싶다. | 가장 합리적인 판매 타이밍을 결정하고, 판매 손실을 최소화할 수 있다. | **[정보 비대칭성]** - 시세 정보의 파편화 |
| **US-2** | 축산 농가 입문자 | 내가 키우는 품종에 특화된 질병 예방 가이드와 처리 과정을 단계별로 알고 싶다. | 즉각적인 대응이 가능하고, 사료/약품 선택에 대한 확신을 얻을 수 있다. | **[전문 지식 접근성]** - 전문 지식의 난해함 |
| **US-3** | 축산 산업 종사자 | 내가 겪는 복합적인 문제를 전문가 그룹과 비공개로 상담하고 해결책을 찾고 싶다. | 단순 질문을 넘어, 동종 업계의 노하우를 공유하고 비즈니스 문제를 해결할 수 있다. | **[지식 공유의 한계]** - 공론장에서의 비체계적 논의 |

### ⚙️ B. 핵심 기능 아이디어 (Core Feature Ideas)

| 기능명 | 목표 기능 (MVP Scope) | 기술 구현 방향 | 경쟁 우위 요소 |
| :--- | :--- | :--- | :--- |
| **1. 실시간 시세 비교 대시보드** | 특정 품종(예: 한우)의 실시간 경매가/도매가 추이를 차트와 표로 한눈에 비교 제공. | 외부 API 연동 (농산물/축산물 시장 데이터), 사용자별 관심 품종 저장 기능. | **'직관적인 비교'**에 초점을 맞춰, 여러 사이트를 거치는 수고를 덜어줌. |
| **2. 품종별 맞춤형 가이드 (Roadmap)** | 품종 선택 $\rightarrow$ 사육 단계(초기/성장/판매) $\rightarrow$ 필수 관리 항목(사료, 질병, 환경) 순으로 체크리스트 제공. | 데이터베이스 기반의 단계별 퀘스트/체크리스트 시스템. | **'체계적인 학습 로드맵'** 제공. 단순 정보 나열이 아닌, 진행 과정을 안내. |
| **3. 전문가 Q&A 매칭 (Premium)** | 일반 Q&A 외에, '유료/전문가 검증'을 거친 비공개 상담 채널을 운영. (혹은 챗봇 초기 버전) | 챗봇(LLM) 연동 및 전문가 검수 프로세스 구축. | **'신뢰성'과 '심화 상담'**을 결합하여, 단순 정보 습득을 넘어 문제 해결을 지원. |
